# Hashlake Effects Contract

- Trade/whale splash is a local event effect and must not drive global weather color.
- BTC amount may scale splash blocks, ripple radius, local foam, local water disturbance, toast text, and Debug values only.
- Global sky darkness, fog, weather band, apocalypse tint, and stormIndex are owned by the weather engine.
- Weather controls global atmosphere.
- Data freshness controls stale fog.
- Whale/trade events control local splash/ripple only.
- New block controls signal pulse/block swell only.
- Boat throttle controls wake only.
- Drive controls boat only.
- Quality governor protects smoothness.
- Event toasts must be visible for user-facing events.
- No effect should silently log only; visual events must be visible.
- Drive Mode camera and coordinate behavior stay independent from visual effects.
- Effects should be capped or pooled before adding new high-frequency visuals.
